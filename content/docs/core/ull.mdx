# üíµ Universal Liquidity Layer

## üåç **Trustless Coordination**

The Universal Liquidity Layer (ULL) finds the optimal bridging path based on speed, cost, and security. You can query the ULL for the best route to bridge tokens between chains.

```javascript
import { useULL } from 'okto-sdk';

// Initialize ULL for cross-chain swaps
const ull = useULL({ chain: 'Ethereum' });

// Swap USDC on Ethereum to BNB on BNB Chain
ull.swap({
  fromToken: 'USDC',
  toToken: 'BNB',
  amount: 100,
  destinationChain: 'BNB Chain',
});
```
The fetchBestRoute function asks the ULL to evaluate different cross-chain solutions like Socket or LayerZero, and return the best path considering factors like transaction fees and speed.
```javascript
import { fetchBestRoute } from 'okto-gateway-sdk';

const route = fetchBestRoute({
  sourceChain: 'Ethereum',
  targetChain: 'BNB Chain',
  token: 'USDC'
});

console.log('Best route:', route);
