### 1. **Interactive Code Editor**
Try out live smart contract coding in our interactive editor. You can write Solidity, Rust, or any Web3-compatible language and deploy directly to a blockchain.

contract HelloWorld {
    string public message = "Welcome to the Okto Web3 Playground!";

        message = newMessage;
    }
