# Playground: Explore and Interact

The Playground is your space to explore and interact with the features of the Okto Layer. Whether you want to see how decentralized wallet networks work or try out cross-chain swaps, you can do it here.

# Cross-Chain Swap

In this example, we'll show you how to swap tokens between two blockchains seamlessly using the Okto Layer's ULL.

1. Select the source and target chains.
2. Enter the token and amount to swap.
3. Watch as the system handles routing, gas fees, and liquidity.
